# /extract-templates

Extract reusable templates from current work.

## Usage

```
/extract-templates
```

## Steps

1. Identify code or documentation patterns that could be reused
2. Generalize the pattern (replace specific names with placeholders)
3. Save to `.agents/templates/` with descriptive name
4. Update relevant AGENTS.md to reference the template

## Template Categories

- `tool-*.md` - MCP tool implementation templates
- `resource-*.md` - MCP resource implementation templates
- `prompt-*.md` - MCP prompt implementation templates
- `test-*.md` - Test templates
